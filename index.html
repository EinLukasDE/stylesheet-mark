Mark element

The Mark element is practically represented in new browsers, unfortunately not for older ones. For this I have found a CSS solution, I make this of course available.


How to use

First, the mark.css file must be included in the header.

      
        <link href="mark.css" rel="stylesheet">

Next you can already use the mark.css.

      
        <p class="mark">Example text</p>
